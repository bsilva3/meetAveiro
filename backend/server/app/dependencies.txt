#instruções#
############

- para o servidor deve ser enviado um JSON do tipo:

{
    "image" : <imagem convertida para base64 string>
}

- o JSON deverá ser enviado como POST para a url 'http://127.0.0.1:8080/search'
- como resposta, o servidor envia o JSON:

{
    "name" : <nome da imagem (string)>
    "description" : <breve descrição sobre a imagem (string)>
}



- para correr o servidor basta correr:

python3 server.py

- há um ficheiro de teste para simular um cliente REST (api_client.py)
- o ficheiro test.py serve apenas para testar o webcrawler
